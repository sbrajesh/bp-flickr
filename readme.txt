Thanks for downloading Bp-Flickr, Hope you will find it Useful.

Installation:-
1.Upload "bp-flickr" to wp-content/plugins
2.Activate Plugin Bp-Flickr from Dashboard->Plugins->Installed->Bp-Flickr
3.Move "flickr" folder to your current theme
4.Customize the css as you want, in the yourtheme/flickr/style.css

That's it.
